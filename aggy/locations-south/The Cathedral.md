# The Cathedral

An impressive chamber accessed via an aven tube and chimney climb from Priory Road.

## Access
- Diversion from Priory Road
- Requires climbing through an aven tube
- Chimney climb to reach the chamber

## Features
- Large chamber with cathedral-like dimensions
- Worth the diversion for the views

## Route Context
Optional side trip on the **Iles Inlet** route.
