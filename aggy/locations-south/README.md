# Locations - South Survey

This folder contains location markers for the **Agen Allwedd South Survey** segment.

## Coverage
This survey covers the southern portions of the cave system including:
- Turkey Streamway
- Sand Caverns / Summertime Series
- Inner and Outer Circle areas
- St Paul's Passage and Dome
- Swiss Village
- Maytime and Resurrection Passage
- Sump 4 area

## Marker Format
Each `.md` file represents a location marker with frontmatter:
```yaml
---
location: [x, y]  # Coordinates on the survey image
mapmarker: type   # chamber, passage, junction, formation, etc.
---
```
